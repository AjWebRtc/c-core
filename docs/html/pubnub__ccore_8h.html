Detailed Description

The "C core" module is the internal module of the Pubnub C clients, shared by all. It has all the "generic" code (formatting, parsing, HTTP...) while individual C clients deal with interacting with the environment (OS, frameworks, libraries...).

The interface of this module is subject to change, so user of a Pubnub C client should not use this module directly. Use the official and documented API for your environment.

enum pubnub_res pbcc_parse_channel_registry_response ( struct pbcc_context *  p)

Parses the string received as a response for a channel-registry operation (transaction). It is done on several user requests (add/remove channel (from/to) channel group, list (channels in a) channel group, remove channel group).

This checks if the response is valid (a JSON object), and, if it is, enables getting it, as a whole, in one pubnub_get().
