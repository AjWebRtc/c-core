Detailed Description

A bunch of functions for parsing JSON. These are designed for internal use by the Pubnub client, but, since they are rather generic, the user can use them, too, though their availability and interface are not guaranteed to survive Pubnub client version changes.

Finds the end of the "complex" value starting from start, until end. Interprets "complex value" as a JSON object or array - that should be compatible with a lot of other specifications.

Assumes that start points to the first character of the complex object (opening curly brace or square bracket).

Returns
Pointer to the character that is the end of the complex. It is == end if the end of the complex was not found in input.

Finds the end of the string starting from start, until end. Interprets string as JSON does (starting and ending with double-quotation and allowing escape characters with backslash) - that should be compatible with a lot of other specifications.
