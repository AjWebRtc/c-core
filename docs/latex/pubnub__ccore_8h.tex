\subsection{Detailed Description}
The \char`\"{}\-C core\char`\"{} module is the internal module of the Pubnub C clients, shared by all. It has all the \char`\"{}generic\char`\"{} code (formatting, parsing, H\-T\-T\-P...) while individual C clients deal with interacting with the environment (O\-S, frameworks, libraries...).

The interface of this module is subject to change, so user of a Pubnub C client should {\itshape not} use this module directly. Use the official and documented A\-P\-I for your environment.
